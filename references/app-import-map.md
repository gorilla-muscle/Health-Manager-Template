# app-import-map.md — アプリ出力ファイルの列マッピング定義

筋トレアプリ・食事記録アプリが出力する CSV / PDF / HTML を取り込む際の、列(項目)→ 本リポジトリのスキーマ対応表。daily-log スキルが取込時に参照し、未登録アプリの初回取込時にユーザー確認を経て追記する。

## 規則

- **本表に登録済みのアプリのみ自動マッピングを適用する。** 未登録アプリの列の意味・単位を推測してはならない(初回はユーザー確認必須)
- マッピングの対応先は profile-schema.md のスキーマ(exercise-log: 日付/種目名/種別/重量kg/レップ/セット/距離km/時間min、metrics: 体重kg/体脂肪率%/睡眠h/歩数/水分ml/摂取kcal/P/F/C)に限定する
- 単位が kg / kcal / g / km / min / ml 以外(lbs, oz, mi 等)の場合は換算式を「単位」列に明記する
- 追記時のコミットメッセージ: `db: add app-import-map <アプリ名>`
