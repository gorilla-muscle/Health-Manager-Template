# ヒアリング質問定義(monthly-plan)

AskUserQuestions で実施する。1回の質問は3問以内(絶対規則4)。以下のラウンド構成で進める。

## ラウンド-1: 対象期間の確認(必須・最初に実施)

計画作成の対象開始日・終了日を確定する。**「今月」= 現在日付〜当月末日 が既定**(月の途中依頼を第一級ケースとして扱う。1日開始を暗黙前提にしない。絶対規則8)。

| # | 質問 | 選択肢 | 備考 |
|---|---|---|---|
| -1-1 | 計画作成の対象期間はどうしますか? | 現在日付から当月末日まで(既定) / 当月1日から末日まで / 開始日と終了日を指定 / 来月1日から末日まで | 選択に応じて開始日 SD・終了日 ED を確定 |

- 「開始日と終了日を指定」が選ばれた場合、追加で自由入力(YYYY-MM-DD 形式)を求める
- 確定後、`SD` と `ED` を以降のヒアリング・計画生成・Issue 発行で使用する
- 対象期間が7日未満の場合は「短期プランになりますが進めますか?」と再確認する
- 対象期間が2か月以上に跨る場合は「monthly-plan は単一の年月単位が前提です。跨月は別スキル対象または月ごと分割で再依頼してください」と案内し停止する

## ラウンド0: 基本情報(data/profile.md が無い場合のみ全問。ある場合は差分確認1問に短縮)

| # | 質問 | 形式 | 備考 |
|---|---|---|---|
| 0-1 | 性別 | 選択: 男性 / 女性 / その他・回答しない | PFC算出の係数に使用 |
| 0-2 | 身長・体重 | 自由入力(cm / kg) | |
| 0-3 | 体脂肪率 | 自由入力(%)、**任意** | 未入力でも処理を止めない(絶対規則5) |

profile.md がある場合の差分確認: 「登録情報は 身長X cm / 体重Y kg / 体脂肪率Z% です。変更はありますか?」(選択: 変更なし / 体重のみ更新 / 全て更新)

## ラウンド1: 目標・トレーニング練度

**このラウンドの2問は profile.md の有無にかかわらず毎月必ず質問する**(練度は月次で更新しうるため profile へ永続化せず、都度確認する)。

| # | 質問 | 選択肢 |
|---|---|---|
| 1-1 | 今月の目標 | ボディメイク大会 / 筋肥大 / 全体的なシェイプアップ / 特定部位の強化 / 持久力向上 |
| 1-2 | トレーニング練度(継続年数) | 初心者(1年未満) / 中級者(1〜3年) / 上級者(4年以上) |

- 練度は種目選定・ボリューム・分割・強度テクニックの調整に使用する(`programming.md` §1.5「練度別の調整」)。回答に沿ってメニューの難易度を構築する
- 練度は毎月このラウンドで再確認する。前月の `hearing.md` に練度の記録がある場合は「前回は{{前回練度}}でした。変更はありますか?」の確認でよい

### 分岐A: 「ボディメイク大会」選択時(追加ラウンド)

| # | 質問 | 形式 |
|---|---|---|
| A-1 | 出場日はいつか | 自由入力(YYYY-MM-DD) |
| A-2 | 現在は増量期か減量期か | 選択: 増量期 / 減量期 / 調整期(大会直前) |

その後、前月の進捗(`data/metrics/` の体組成推移・前月 summary.md の計画達成率)を読み込み、大会日から逆算した当月フェーズ(例: 「大会まで10週 → 減量中盤。週あたり体重-0.5%ペース」)を提示して合意を取る。

**安全ゲート**: 減量ペースが月間体重の5%を超える場合は計画生成を保留し、専門家への相談を促す(絶対規則6)。

### 分岐B: 「特定部位の強化」選択時

| # | 質問 | 選択肢 |
|---|---|---|
| B-1 | 強化したい部位 | 胸 / 背中 / 脚 / 肩 / 腕 / 腹・体幹 |

### 分岐C: 減量が目的の場合(「全体的なシェイプアップ」選択時、または「ボディメイク大会」×「減量期」選択時)

減量の進め方(減量方法)を確認する。摂取カロリーは §4.3 の設定(TDEE −10〜20%)で共通、たんぱく質(P)は減量方法によらず維持し、**脂質(F)と炭水化物(C)の配分**が方法ごとに変わる(`programming.md` §4.4a)。

| # | 質問 | 選択肢 |
|---|---|---|
| C-1 | 減量の進め方(減量方法)はどうしますか? | 標準(バランス型) / ローファット(低脂質) / ケトジェニック |

各選択肢は必ず以下の説明を添えて提示する(AskUserQuestions の各選択肢の説明文に反映する)。

- **標準(バランス型)**: PFC をバランスよく配分する一般的な減量。脂質は総エネルギーの 20〜30%、炭水化物は残余で確保する。特別な食事制限がなく自由度が高いため継続しやすい。減量方法を特に決めていない場合の既定。
- **ローファット(低脂質)**: 脂質を最低限(総エネルギーの 15〜20%。ただし体重×0.5 g/日は下回らない)まで抑え、浮いた分を炭水化物に回す方法。炭水化物を多めに確保できるためトレーニング強度を維持しやすい。揚げ物・油脂の多い食材を控える食事になる。
- **ケトジェニック**: 炭水化物を 1 日 20〜50 g(総エネルギーの約 5〜10%)まで厳しく制限し、脂質を主エネルギー源に切り替える方法(体を脂質燃焼=ケトーシスの状態にする)。効果に個人差が大きく、導入初期に倦怠感(いわゆるケトフルー)が出ることがある。脂質摂取量が多いため、脂質の消化・分解が苦手な体質には向かない。

**体質特性との整合(絶対規則9)**: `data/profile.md` の体質特性に「脂質分解が苦手」「低脂質適応」など**脂質上限(F 上限)**が登録されている場合、ケトジェニック(高脂質が前提)は矛盾する。この場合はユーザーに矛盾を提示し、**標準またはローファットを推奨する**。体質特性を維持したままケトジェニックを自動採用しない(健康影響の回避・絶対規則6)。体質特性そのものを見直したい場合は profile-manager で承認のうえ更新してから再選択する。

**安全ゲート(絶対規則6)**: いずれの減量方法でも、摂取カロリーの下限(BMR 未満は不可)と減量ペースの上限(月間体重の 5%)は §4.3 の規則を優先する。減量方法の選択でこれらの安全規則を緩めない。

## 記録

全回答は `plans/YYYY-MM/hearing.md` に以下の形式で保存する。

```markdown
# YYYY-MM ヒアリング控え
- 実施日: YYYY-MM-DD
- 対象期間: YYYY-MM-DD 〜 YYYY-MM-DD(N日間)※途中開始の場合はその旨明記
- 基本情報: 性別 / 身長 / 体重 / 体脂肪率(未入力なら「未入力」)
- 目標: (分岐回答含む)
- 減量方法(減量が目的の場合のみ): 標準(バランス型) / ローファット / ケトジェニック(未該当なら「該当なし」)
- トレーニング練度: 初心者 / 中級者 / 上級者
- 頻度・時間・場所:
- 食事スタイル:
- 安全チェック: 問題なし / 保留(理由)
```
