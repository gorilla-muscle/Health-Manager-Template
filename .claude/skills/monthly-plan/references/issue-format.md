# 日次 Issue 本文テンプレート

monthly-plan が発行する日次 Issue の本文形式。`{{ }}` を実値に置換する。

## 重要: Issue はユーザー向け成果物

Issue 本文はスマホ・PC でユーザーが直接読む。**アーティファクト(HTML)と同等の表現制約**を適用する(CLAUDE.md「読者別の表現使い分け」参照)。特に以下を守る:

- **内部ファイル名・パス・セクション番号を Issue 本文に書かない**
  - NG: `plans/2026-07/plan.md §4`・`programming.md §6`・`references/exercise-db.md`・`nutrition-db.md`・`log-format.md`
  - NG: `絶対規則3`・`§4.3 に準拠`・`programming.md §4.4 の override`
  - OK: 「バランスを調整した1日パターン」「体質に合わせた低脂質設計」「経験者向けの重量目安」等の一般語で言い直す
- **英略語(BMR / TDEE / PFC / RPE / Epley / 1RM 等)は初出時に日本語補足を必ず付ける**
  - 例: 「主要栄養素の内訳(PFC=たんぱく質・脂質・炭水化物)」「主観的運動強度(RPE・1〜10で今日どれだけキツかったか)」
- **種目メモは動作イメージが浮かぶ書き方にする**(専門解剖学用語を並べない)
- **重量は必ず"目安"を添える**。ユーザーは"自分の重さ"の見当がつかないと止まる

## タイトル

- トレーニング日: `[{{YYYY-MM-DD}}] Day{{NN}} - {{分割名}}({{部位}}メイン)`
  - **対象部位は必ず「〜メイン」で表現する**(「主眼」「主働」等の硬い言い回しは使わない)
  - OK 例: `Push A(胸メイン)` / `Pull A(背中メイン)` / `Legs A(脚メイン)` / `Push B(肩メイン)` / `Arms(腕メイン)` / `有酸素+腹(積極的休養)`
  - NG 例: `Push A(胸主眼)` / `Pull B(背中厚み・二頭)` / `Legs B(ハム・臀部)` — 細分名を並べず「そのセッションで主に狙う部位1つ」を選ぶ
- 休養日(トレ0日の完全休養): `[{{YYYY-MM-DD}}] Day{{NN}} - Rest`

## ラベル

- トレーニング日: `training` `nutrition` `month:{{YYYY-MM}}`
- 休養日: `rest` `nutrition` `month:{{YYYY-MM}}`

## フォーマット規則

- **本日のメニューの並び順(固定)**: トレーニング日は メトリクス目標 → 食事目標 → トレーニング目標、休養日は メトリクス目標 → 食事目標 の順で記載する。この順序を変えない
- **数値根拠は内部で確認、Issue には書かない**:
  - メトリクス目標(歩数・睡眠・水分)や PFC の算出根拠(programming.md §4・§6)はスキル実行時に**エージェント側で参照**する
  - Issue 本文には「体重1kg あたり 35ml が目安」等の**平易な理由**のみを書く(ファイル名・§番号は書かない)
- **体調チェック**: GitHub のタスクリスト記法(`- [ ]`)で記載する。チェック済み(`- [x]`)は「問題あり」を意味する。ユーザーが記入する欄なので発行時は未チェックのままにする
- **セクション間隔**: 可読性のため、各セクション(見出し・表・メモブロック)の区切りに `<br>` を1行単独で挿入する。`<br>` の前後には空行を置く(GitHub のレンダリングで確実に間隔が入る)
- 見出し直後のコンテンツとの間には空行を入れる(CommonMark 準拠)
- Issue 本文を生成・更新する際は本テンプレートの `<br>` 配置をそのまま踏襲する

## Issue タイトルの「〜メイン」対応表(参考)

分割別の**推奨タイトル部位名**の例。プログラムで決めない・エージェントが分割の狙いに合わせて選ぶ。

| 分割 | 主な狙い | 推奨タイトル部位名 |
|---|---|---|
| Push A(胸コンパウンド中心) | 大胸筋 | 胸メイン |
| Push B(肩コンパウンド中心) | 三角筋 | 肩メイン |
| Pull A(垂直プル中心=懸垂・ラットプル) | 広背筋(幅) | 背中メイン |
| Pull B(水平プル中心=ロー系) | 広背筋(厚み)・僧帽 | 背中メイン |
| Legs A(四頭中心=スクワット系) | 大腿四頭 | 脚メイン |
| Legs B(ヒップヒンジ中心=RDL・ヒップスラスト) | ハム・臀部 | 脚メイン |
| Arms(腕独立日) | 二頭・三頭 | 腕メイン |
| Full Body(全身1日) | 大筋群全般 | 全身メイン |
| 有酸素+腹(週次挿入日) | 心肺・体幹 | 積極的休養 |
| 完全休養(トレ0日) | — | (タイトルは `Rest` 単独) |

Pull A/B が両方「背中メイン」になる等、分割違いでタイトル部位が重複することはある(問題ない・サブラベルの `A`/`B` で区別する)。
