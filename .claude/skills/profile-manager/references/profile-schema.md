# プロフィール・メトリクス スキーマ定義

単位は全ファイル共通で固定: 体重 **kg** / 体脂肪率 **%** / 睡眠 **時間(h)** / 歩数 **歩** / 水分 **ml** / エネルギー **kcal** / PFC **g**。

## 1. data/profile.md

```markdown
# プロフィール

- 更新日: YYYY-MM-DD

| 項目 | 値 |
|---|---|
| 性別 | 男性 / 女性 / その他・回答しない |
| 生年(年齢算出用・任意) | YYYY |
| 身長 | XXX.X cm |
| 体重 | XX.X kg |
| 体脂肪率(任意) | XX.X % または — |

## 目標(現在)

- 目標: (monthly-plan のヒアリング結果を転記)
- 大会予定日(該当時のみ): YYYY-MM-DD

## 更新履歴

| 日付 | 項目 | 旧値 | 新値 |
|---|---|---|---|
```

## 2. data/metrics/YYYY-MM.md(1行=1日)

```markdown
# YYYY-MM メトリクス

| 日付 | 体重kg | 体脂肪率% | 睡眠h | 歩数 | 水分ml | 摂取kcal | P | F | C | 備考 |
|---|---|---|---|---|---|---|---|---|---|---|
| YYYY-MM-DD | 70.2 | 18.5 | 7.0 | 8500 | 2000 | 2150 | 145 | 60 | 230 |  |
```

- 未計測項目は「—」(空欄にしない。列ずれ防止)
- 体脂肪率は任意項目
- 日付は昇順を維持する

## 3. data/exercise-log/YYYY-MM.md(1行=1種目実績)

```markdown
# YYYY-MM 種目別実績ログ

| 日付 | 種目名 | 種別 | 重量kg | レップ | セット | 距離km | 時間min | 備考 |
|---|---|---|---|---|---|---|---|---|
| YYYY-MM-DD | バーベルベンチプレス | strength | 60 | 8 | 3 |  |  | RPE8 |
| YYYY-MM-DD | ランニング | cardio |  |  |  | 5.2 | 31 |  |
```

- 種別は `strength` / `cardio` の2値
- strength: 距離km・時間min は空欄 / cardio: 重量kg・レップ・セットは空欄
- 種目名は `references/exercise-db.md` の正式名称と完全一致させる
- 本ファイルが増減率グラフの**唯一の計算ソース**(Issue 再パース禁止)
```
